# PetProjects
1. OptionsGame.py - Use Black-Scholes valuation to play against an AI bot in placing calls/puts based on news flashes. The AI is trained using reinforcement learning (QLearning), before (10k episodes) and during the game.
2. CableForecasting.py - Trains deep learning neural networks over a 24 month window to model cable over a 6-month horizon, testing it against actual observed data. Models use DXY and EUR/USD as covariates.
